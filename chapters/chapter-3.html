Section 1 of 13

Matrix Notation

An \(m\times n\) matrix is a rectangular array with \(m\) rows and \(n\) columns. It is written \(A=(a_{ij})\), where \(a_{ij}\) denotes the entry in row \(i\), column \(j\).

DefinitionThe dimensions \(m\times n\) determine which matrix operations are defined. Rows are indexed by \(i\), columns by \(j\), and entries are written \(a_{ij}\) or \(A_{ij}\).
Important link
The first subscript is always the row number. The second subscript is always the column number.
For \(A_{ij}\), read \(i\) first, then \(j\).
Course checklist
\(A=(a_{ij})_{m\times n}\) means \(1\le i\le m\) and \(1\le j\le n\).
The entry \(a_{ij}\) is a scalar, while \(A\) is the whole matrix.
A square matrix has the same number of rows and columns.
\(A_{21}\) means row \(2\), column \(1\).
TRY Entry reading
For \(A=\begin{bmatrix}1&3&5&7\\9&6&3&0\\-2&-4&-8&\sqrt2\end{bmatrix}\), find \(A_{34}\).
Row \(3\) is \((-2,-4,-8,\sqrt2)\).Column \(4\) is the last entry of that row.\(A_{34}=\sqrt2\).
Section 2 of 13

Matrix Types

Matrix types are classified by dimensions and by restrictions on their entries, such as zero entries outside a diagonal.

Course checklist
A row matrix has size \(1\times n\). A column matrix has size \(m\times1\).
A scalar matrix is a diagonal matrix whose diagonal entries are all equal.
The identity matrix \(I_n\) is the scalar matrix with every diagonal entry equal to \(1\).
Row matrix
\(\begin{bmatrix}a_1&a_2&\cdots&a_n\end{bmatrix}\)
Column matrix
\(\begin{bmatrix}b_1\\b_2\\\vdots\\b_m\end{bmatrix}\)
Identity matrix
\(I_3=\begin{bmatrix}1&0&0\\0&1&0\\0&0&1\end{bmatrix}\)
Diagonal matrix
\(\begin{bmatrix}a&0&0\\0&e&0\\0&0&g\end{bmatrix}\)
Upper triangular
\(\begin{bmatrix}a&b&c\\0&e&f\\0&0&g\end{bmatrix}\)
Lower triangular
\(\begin{bmatrix}a&0&0\\c&d&0\\e&f&g\end{bmatrix}\)
Zero matrixThe matrix \(0\) has every entry equal to zero. Its size depends on the context.
TRY Classify
Classify \(B=\begin{bmatrix}3&0&0&0\\0&3&0&0\\0&0&0&0\\0&0&0&\pi\end{bmatrix}\).
All entries away from the main diagonal are zero.\(B\) is diagonal. It is also square.
Section 3 of 13

Equality & Entry Rules

Two matrices are equal exactly when they have the same dimensions and all corresponding entries are equal.

Equality testIf \(A\) and \(B\) are both \(m\times n\), then \(A=B\) means \(A_{ij}=B_{ij}\) for every valid \(i,j\).
Course checklist
Entrywise comparison is valid only after confirming that the two matrices have the same size.
A matrix equation often becomes several scalar equations, one from each matching entry.
EXAMPLE 1 Matrix equation
Solve for \(a,b,c,d\): \[ \begin{bmatrix}a-b&b+c\\3d+c&2a-4d\end{bmatrix} = \begin{bmatrix}8&1\\7&6\end{bmatrix}. \]
Match entries: \(a-b=8,\ b+c=1,\ 3d+c=7,\ 2a-4d=6\).From \(a-b=8\), \(b=a-8\). From \(b+c=1\), \(c=9-a\).Then \(d=(7-c)/3=(a-2)/3\).Use \(2a-4d=6\): \(2a-\frac{4(a-2)}3=6\), so \(a=5\).\(a=5,\ b=-3,\ c=4,\ d=1\).
TRY Build from a rule
Let \(B=[b_{ij}]\) be \(3\times3\), where \(b_{ij}=0\) if \(i=j\), \(b_{ij}=i+j\) if \(i\gt j\), and \(b_{ij}=-j\) if \(i\lt j\). Find \(B\).
Diagonal entries are \(0\).Below the diagonal use \(i+j\). Above the diagonal use \(-j\).\(B=\begin{bmatrix}0&-2&-3\\3&0&-3\\4&5&0\end{bmatrix}\).
Section 4 of 13

Addition & Scalar Multiplication

Matrix addition and subtraction are defined only for matrices of the same size and are computed entrywise. Scalar multiplication is defined for every matrix.

RulesFor same sized matrices, \((A+B)_{ij}=A_{ij}+B_{ij}\) and \((A-B)_{ij}=A_{ij}-B_{ij}\). For a scalar \(\alpha\), \((\alpha A)_{ij}=\alpha A_{ij}\).
Important link
Same size means the same number of rows and the same number of columns.
Scalar multiplication never changes the size of the matrix.
Warning Matrices of different sizes cannot be added or subtracted.
EXAMPLE 2 Entry arithmetic
Let \[ A=\begin{bmatrix}4&3&2\\1&3&1\end{bmatrix},\quad B=\begin{bmatrix}7&2&0\\-5&3&-1\end{bmatrix},\quad C=\begin{bmatrix}3&-6&9\\3&0&12\end{bmatrix}. \] Find \(2A-B+\frac13C\).
\(2A=\begin{bmatrix}8&6&4\\2&6&2\end{bmatrix}\), and \(\frac13C=\begin{bmatrix}1&-2&3\\1&0&4\end{bmatrix}\).Subtract \(B\), then add \(\frac13C\).\(2A-B+\frac13C=\begin{bmatrix}2&2&7\\8&3&7\end{bmatrix}\).
TRY Size check
Can \(\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}+\begin{bmatrix}7&8\\9&10\end{bmatrix}\) be computed?
The first matrix is \(2\times3\). The second is \(2\times2\).No. Addition needs the same size.
Section 5 of 13

Matrix Multiplication

If \(A\) is \(m\times r\) and \(B\) is \(r\times n\), then \(AB\) is defined and has size \(m\times n\). Each entry is computed from one row of \(A\) and one column of \(B\).

Product ruleIf \(A\) is \(m\times r\) and \(B\) is \(r\times n\), then \(AB\) is \(m\times n\), and \((AB)_{ij}=\sum_{k=1}^{r}A_{ik}B_{kj}\).
Important link
The inner dimensions decide whether multiplication is defined. The outer dimensions give the size of the answer.
Matrix multiplication is not usually commutative, so \(AB\) and \(BA\) can be different or one may not exist.
Course checklist
The entry \((AB)_{ij}\) uses row \(i\) of \(A\) and column \(j\) of \(B\).
The summation index \(k\) runs over the shared inner dimension.
Even when both \(AB\) and \(BA\) are defined, they need not be equal.
\((AD)_{11}=(1,2,-1)\cdot(-2,4,2)=4\).
EXAMPLE 3 Product and size
Let \(A=\begin{bmatrix}1&2&-1\\3&1&4\end{bmatrix}\) and \(B=\begin{bmatrix}-2\\4\\2\end{bmatrix}\). Find \(AB\).
The sizes are \(2\times3\) and \(3\times1\), so \(AB\) is \(2\times1\).First entry: \(1(-2)+2(4)+(-1)(2)=4\).Second entry: \(3(-2)+1(4)+4(2)=6\).\(AB=\begin{bmatrix}4\\6\end{bmatrix}\).
TRY Order matters
For the \(A\) and \(B\) above, is \(BA\) defined?
\(B\) is \(3\times1\), and \(A\) is \(2\times3\).The inner sizes would be \(1\) and \(2\), which do not match.\(BA\) is not defined.
Section 6 of 13

Identity & Transpose

The identity matrix \(I_n\) is the \(n\times n\) matrix with ones on the main diagonal and zeros elsewhere. The transpose \(A^T\) swaps rows and columns.

Course checklist
The main diagonal consists of entries \(a_{11},a_{22},\ldots\).
Transposing an \(m\times n\) matrix gives an \(n\times m\) matrix.
The identity matrix used on the left and right may have different sizes when \(A\) is not square.
Identity
If \(A\) is \(m\times n\), then \(AI_n=A\) and \(I_mA=A\).
Transpose
\((A^T)_{ij}=A_{ji}\)
Transpose laws\((A^T)^T=A\), \((A+B)^T=A^T+B^T\), \((\alpha A)^T=\alpha A^T\), and \((AB)^T=B^TA^T\).
Important link
The identity matrix size must match the multiplication side: \(AI_n=A\) and \(I_mA=A\) for an \(m\times n\) matrix \(A\).
The transpose of a product reverses the order: \((AB)^T=B^TA^T\).
EXAMPLE 4 Transpose
Find \(A^T\) for \(A=\begin{bmatrix}4&3&2\\1&3&1\end{bmatrix}\).
Rows of \(A\) become columns of \(A^T\).\(A^T=\begin{bmatrix}4&1\\3&3\\2&1\end{bmatrix}\).
TRY Product transpose
If \(A\) is \(2\times3\) and \(B\) is \(3\times4\), what is the size of \((AB)^T\)?
\(AB\) is \(2\times4\).Taking transpose swaps the size.\((AB)^T\) is \(4\times2\).
Section 7 of 13

Inverse Matrices

A square matrix \(A\) is invertible if a matrix \(A^{-1}\) exists such that \(AA^{-1}=I\) and \(A^{-1}A=I\). Non square matrices do not have two sided inverses of this type.

DefinitionA square matrix \(A\) is invertible if there is a square matrix \(B\) such that \(AB=I\) and \(BA=I\). The inverse is unique and is written \(A^{-1}\).
Important link
\(A^{-1}\) is the inverse matrix, not the reciprocal of every entry.
A matrix with determinant \(0\) is singular, so its inverse does not exist.
Course checklist
Only square matrices can be invertible in the two sided sense used here.
If \(A^{-1}\) exists, then \(AA^{-1}=A^{-1}A=I_n\).
A matrix that is not invertible is called singular.
Singular clues
  • A row of zeros
  • A column of zeros
  • One row is a multiple of another
  • A row is a linear combination of other rows
2 by 2 formula
For \(A=\begin{bmatrix}a&b\\c&d\end{bmatrix}\), if \(ad-bc\ne0\), then \(A^{-1}=\dfrac1{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}\).
EXAMPLE 5 Check invertibility
Determine whether \(A=\begin{bmatrix}5&3\\7&9\end{bmatrix}\) is invertible. If it is, find \(A^{-1}\).
Compute \(ad-bc=5(9)-3(7)=24\ne0\).So \(A\) is invertible.\(A^{-1}=\frac1{24}\begin{bmatrix}9&-3\\-7&5\end{bmatrix}\).
TRY Singular or invertible
Is \(B=\begin{bmatrix}-8&4\\6&-3\end{bmatrix}\) invertible?
\(ad-bc=(-8)(-3)-4(6)=24-24=0\).\(B\) is singular.
Section 8 of 13

Diagonal Matrices & Powers

A diagonal matrix has possible nonzero entries only on the main diagonal. A triangular matrix has all entries zero on one side of the main diagonal.

Course checklist
Matrix powers \(A^n\) require \(A\) to be square.
\(A^0=I_n\) for an \(n\times n\) matrix \(A\).
Negative powers such as \(A^{-2}\) require \(A\) to be invertible.
Diagonal inverse
\(\operatorname{diag}(a,b,c,d)^{-1}=\operatorname{diag}\!\left(\frac1a,\frac1b,\frac1c,\frac1d\right)\)
Power notation
\(A^0=I,\quad A^n=AA\cdots A,\quad A^{-n}=(A^{-1})^n\)
Triangular factA triangular matrix is invertible exactly when every diagonal entry is non zero. Its determinant is the product of its diagonal entries.
Important link
Matrix powers such as \(A^2\) and \(A^{-1}\) require \(A\) to be square.
For a diagonal matrix, powers and inverses act entry by entry on the diagonal, when the needed entries are nonzero.
EXAMPLE 6 Matrix powers
Let \(A=\begin{bmatrix}1&1\\0&1\end{bmatrix}\). Find \(A^2,A^3,A^{-2}\).
Multiplying shows \(A^n=\begin{bmatrix}1&n\\0&1\end{bmatrix}\) for positive integers \(n\).Also \(A^{-1}=\begin{bmatrix}1&-1\\0&1\end{bmatrix}\).\(A^2=\begin{bmatrix}1&2\\0&1\end{bmatrix},\quad A^3=\begin{bmatrix}1&3\\0&1\end{bmatrix},\quad A^{-2}=\begin{bmatrix}1&-2\\0&1\end{bmatrix}\).
TRY Diagonal inverse
Find the inverse of \(\begin{bmatrix}2&0&0\\0&-5&0\\0&0&\frac13\end{bmatrix}\).
Invert each diagonal entry.\(\begin{bmatrix}\frac12&0&0\\0&-\frac15&0\\0&0&3\end{bmatrix}\).
Section 9 of 13

Determinants & Cofactors

The determinant is a scalar defined only for square matrices. It tests invertibility and appears in area, volume, inverse, and system solving formulas.

Base cases\(\det([a])=a\), and \(\det\begin{bmatrix}a&b\\c&d\end{bmatrix}=ad-bc\).
Important link
Only square matrices have determinants.
A zero determinant means the matrix is not invertible. A nonzero determinant means the matrix is invertible.
Course checklist
The determinant of a triangular matrix is the product of its diagonal entries.
If two rows are equal or one row is a scalar multiple of another, the determinant is \(0\).
The determinant of a product satisfies \(\det(AB)=\det(A)\det(B)\) for square matrices of the same size.
\(C_{11}=(-1)^{1+1}\det\begin{bmatrix}2&1\\2&1\end{bmatrix}=0\).
CofactorThe minor \(M_{ij}\) is the determinant left after deleting row \(i\) and column \(j\). The cofactor is \(C_{ij}=(-1)^{i+j}M_{ij}\).
TRY Cofactor
For \(A=\begin{bmatrix}1&5&0\\-3&2&1\\1&2&1\end{bmatrix}\), find \(C_{31}\).
Delete row \(3\), column \(1\): \(\begin{bmatrix}5&0\\2&1\end{bmatrix}\).Its determinant is \(5(1)-0(2)=5\).The sign is \((-1)^{3+1}=+1\).\(C_{31}=5\).
Section 10 of 13

Cofactor Expansion

Cofactor expansion computes the determinant of a square matrix by expanding along a chosen row or column. Choosing a row or column with zeros reduces the arithmetic.

ExpansionAlong row \(i\), \(\det(A)=a_{i1}C_{i1}+a_{i2}C_{i2}+\cdots+a_{in}C_{in}\). Along column \(j\), \(\det(A)=a_{1j}C_{1j}+\cdots+a_{nj}C_{nj}\).
Important link
A minor deletes one row and one column. A cofactor is a signed minor.
The sign pattern alternates, starting with plus in the top left entry.
Checkerboard signs
\(\begin{bmatrix}+&-&+\\-&+&-\\+&-&+\end{bmatrix}\)
Special determinant facts
  • A zero row gives determinant \(0\)
  • Multiple rows give determinant \(0\)
  • Triangular determinant is diagonal product
  • \(\det(AB)=\det(A)\det(B)\)
EXAMPLE 7 Expand along a row
Find \[ \det\begin{bmatrix}1&5&0\\-3&2&1\\1&2&1\end{bmatrix}. \]
Expand along row \(3\), with signs \(+,-,+\).\(\det(A)=1\begin{vmatrix}5&0\\2&1\end{vmatrix}-2\begin{vmatrix}1&0\\-3&1\end{vmatrix}+1\begin{vmatrix}1&5\\-3&2\end{vmatrix}\).This is \(5-2+17\).\(\det(A)=20\).
TRY Use zeros
Compute \[ \begin{vmatrix}3&1&2&2\\1&0&-2&1\\1&0&5&-1\\2&0&0&1\end{vmatrix}. \]
Expand down column \(2\), since it has three zeros.Only \(a_{12}=1\) contributes. Its sign is \((-1)^{1+2}=-1\).\(\det(A)=-\begin{vmatrix}1&-2&1\\1&5&-1\\2&0&1\end{vmatrix}\).The \(3\times3\) determinant is \(1\).\(\det(A)=-1\).
Section 11 of 13

Adjoint & Invertibility

The adjoint of \(A\) is the transpose of the cofactor matrix. If \(\det(A)\ne0\), it gives the inverse formula \(A^{-1}=\frac1{\det(A)}\operatorname{adj}(A)\).

AdjointThe cofactor matrix is \(C=[C_{ij}]\). The adjoint is \(\operatorname{adj}(A)=C^T\).
Course checklist
The adjoint is built from cofactors, then transposed.
\(\operatorname{adj}(A)\) is not usually the inverse by itself.
The formula \(A^{-1}=\frac1{\det(A)}\operatorname{adj}(A)\) requires \(\det(A)\ne0\).
Main identity
\(A\operatorname{adj}(A)=\det(A)I\)
Inverse test
\(A\) is invertible exactly when \(\det(A)\ne0\).
Inverse formulaIf \(\det(A)\ne0\), then \(A^{-1}=\dfrac1{\det(A)}\operatorname{adj}(A)\).
EXAMPLE 8 Inverse by adjoint
For \(A=\begin{bmatrix}1&5&0\\-3&2&1\\1&2&1\end{bmatrix}\), use \[ \operatorname{adj}(A)=\begin{bmatrix}0&-5&5\\4&1&-1\\-8&3&17\end{bmatrix},\quad \det(A)=20. \] Find \(A^{-1}\).
Use \(A^{-1}=\frac1{\det(A)}\operatorname{adj}(A)\).\(A^{-1}=\frac1{20}\begin{bmatrix}0&-5&5\\4&1&-1\\-8&3&17\end{bmatrix}\).
TRY Determinant of inverse
If \(\det(A)=5\), what is \(\det(A^{-1})\)?
Since \(AA^{-1}=I\), \(\det(A)\det(A^{-1})=\det(I)=1\).\(\det(A^{-1})=\frac15\).

Cramer's Rule & Practice

Cramer's rule applies to a square system \(Ax=b\) when \(\det(A)\ne0\). It expresses each unknown as a ratio of determinants.

Cramer's ruleIf \(A\) is \(n\times n\) and \(\det(A)\ne0\), then the unique solution of \(Ax=b\) is \(x_j=\dfrac{\det(A_j)}{\det(A)}\), where \(A_j\) replaces column \(j\) of \(A\) by \(b\).
Important link
Cramer's rule needs a square coefficient matrix and a nonzero determinant.
\(A_j\) means replace only column \(j\) of \(A\) by the constants column \(b\).
Course checklist
The vector \(b\) is the constants column on the right side of the system.
If \(\det(A)=0\), Cramer's rule does not apply.
When \(\det(A)\ne0\), the system has exactly one solution.
\(\det(A)=-132\). For \(x\), replace column \(1\): \(x=\frac{-36}{-132}=\frac3{11}\).
EXAMPLE 9 Cramer's rule
Solve \[ 4x+5y=2,\quad 11x+y+2z=3,\quad x+5y+2z=1 \] using Cramer's rule.
The coefficient matrix is \(A=\begin{bmatrix}4&5&0\\11&1&2\\1&5&2\end{bmatrix}\), and \(\det(A)=-132\ne0\).\(\det(A_x)=-36\), \(\det(A_y)=-24\), \(\det(A_z)=12\).Divide each by \(\det(A)\).\(x=\frac3{11},\ y=\frac2{11},\ z=-\frac1{11}\).
PRACTICE 1 Matrix product
Let \(A=\begin{bmatrix}3&0\\-1&2\\1&1\end{bmatrix}\) and \(B=\begin{bmatrix}4&-1\\0&2\end{bmatrix}\). Is \(BA\) defined?
\(B\) is \(2\times2\), and \(A\) is \(3\times2\).The inner sizes are \(2\) and \(3\), which do not match.\(BA\) is not defined.
PRACTICE 2 Inverse with parameter
For \(A=\begin{bmatrix}2&k-2\\3&k\end{bmatrix}\), state when \(A\) is invertible.
Compute \(ad-bc=2k-3(k-2)=6-k\).The matrix is invertible when this is not zero.\(A\) is invertible when \(k\ne6\).
PRACTICE 3 Cofactor expansion
For \[ A=\begin{bmatrix}2&1&3&3\\1&0&1&1\\0&2&2&0\\0&1&2&3\end{bmatrix}, \] evaluate \(\det(A)\).
Expanding carefully, the determinant is \(-2\).A good route is to expand along the first column, since it has two zeros.\(\det(A)=-2\).
PRACTICE 4 Reciprocal substitution
Solve \[ \frac1x+\frac2y+\frac1z=1,\quad \frac3x+\frac4y+\frac1z=5,\quad \frac8x+\frac6y+\frac7z=0. \]
Let \(X=\frac1x\), \(Y=\frac1y\), and \(Z=\frac1z\).Solve \(\begin{bmatrix}1&2&1\\3&4&1\\8&6&7\end{bmatrix}\begin{bmatrix}X\\Y\\Z\end{bmatrix}=\begin{bmatrix}1\\5\\0\end{bmatrix}\).This gives \(X=1,\ Y=1,\ Z=-2\).\(x=1,\ y=1,\ z=-\frac12\).

Study Sheet

This revision card lists the matrix operations, dimension checks, determinant tests, and inverse conditions used in the chapter.

Chapter strategy map
Add or subtract?Check that the matrices have the same dimensions.
Multiply A then B?Check that columns of A equal rows of B.
Find determinant?Use only for a square matrix.
Find inverse?Require a square matrix with nonzero determinant.
Solve Ax = b?Use inverse or Cramer's rule only when the determinant is nonzero.
Use adjoint?Build cofactors, transpose, then divide by the determinant.
Decision guide
Use size checksbefore addition, subtraction, multiplication, determinants, and inverses.
Use row by column productsto calculate each entry of \(AB\).
Use determinantsonly for square matrices, especially to test invertibility.
Use adjoint inversefor a square matrix with \(\det(A)\ne0\).

Common mistakes
Size check

Matrices of different sizes cannot be added or subtracted.

Order matters

Usually \(AB\ne BA\), even when both products exist.

Square only

Determinants and inverses are defined only for square matrices.

Adjoint step

The adjoint is the transpose of the cofactor matrix, not the cofactor matrix itself.

5 minute self check
1. When is \(AB\) defined?
When columns of \(A\) equal rows of \(B\).
2. What size is \(AB\) if \(A\) is \(m\times n\) and \(B\) is \(n\times p\)?
The product is \(m\times p\).
3. When does \(A^{-1}\) exist?
When \(A\) is square and \(\det(A)\ne0\).
4. What is \(\det\begin{bmatrix}2&5\\1&3\end{bmatrix}\)?
\(2\cdot3-5\cdot1=1\).
5. What condition does Cramer's rule need?
\(\det(A)\ne0\), so the system has a unique solution.
