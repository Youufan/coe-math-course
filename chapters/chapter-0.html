Sets & Intervals

A set is a collection of distinct elements. An interval is a set of real numbers between specified endpoints; brackets state whether endpoints are included.

Key pointOrder does not matter: \(\{1,2,3\}=\{3,1,2\}\). Use \(x\in A\) for "x is in \(A\)", \(x\notin A\) for "x is not in \(A\)", and \(A\subseteq B\) for "\(A\) is a subset of \(B\)".
Number families
SetMeaning
\(\mathbb{N}\)\(0,1,2,3,\ldots\)
\(\mathbb{Z}\)\(\ldots,-2,-1,0,1,\ldots\)
\(\mathbb{Q}\)fractions \(p/q\)
\(\mathbb{R}\)all real decimals
\(\mathbb{N}\subseteq\mathbb{Z}\subseteq\mathbb{Q}\subseteq\mathbb{R}\)

Intervals

Endpoint ruleA round bracket means the endpoint is excluded. A square bracket means the endpoint is included. Infinity is never an endpoint, so it always uses a round bracket.

Set Operations

Set operations form new sets from existing sets. Union includes elements in either set, intersection includes elements common to both, and difference removes elements of one set from another.

\(A\cup B\): elements in \(A\), in \(B\), or in both.
EX 1 Union and intersection
\(A=\{x\in\mathbb{N}:x\lt 10\}\), \(B=\{x\in\mathbb{R}:2\lt x\le5\}\). Find \(A\cup B\) and \(A\cap B\).
\(A=\{0,1,2,\ldots,9\}\); \(B=(2,5]\).\(A\cap B=\{3,4,5\}\).\(A\cup B=\{0,1,2\}\cup(2,5]\cup\{6,7,8,9\}\).
EX 2 Complements
For \(A=\{x\in\mathbb{N}:x\lt 10\}\), find \(A^c\) in \(\mathbb{N}\) and in \(\mathbb{R}\).
Universe \(\mathbb{N}\): \(\{10,11,12,\ldots\}\).Universe \(\mathbb{R}\): \(\mathbb{R}\setminus\{0,1,2,\ldots,9\}\). Complements depend on the universe.
EX 3 Cartesian products
Describe these sets: 1) \(\mathbb R^2\) 2) \(\mathbb R^3\) 3) \(\mathbb R\times\mathbb Z\).
1) \(\mathbb R^2=\mathbb R\times\mathbb R=\{(x,y):x,y\in\mathbb R\}\). This is the coordinate plane.2) \(\mathbb R^3=\{(x,y,z):x,y,z\in\mathbb R\}\). This is three dimensional coordinate space.3) \(\mathbb R\times\mathbb Z=\{(x,n):x\in\mathbb R,\ n\in\mathbb Z\}\). In the plane, this is all horizontal lines with integer \(y\) values.

Functions, Domain & Range

A function \(f:A\to B\) assigns each input \(x\in A\) exactly one output \(f(x)\in B\). The domain is the set of allowed inputs.

One input gives one output.
Three setsThe domain is the input set. The codomain is the declared target set. The range is the subset of the codomain actually attained by \(f(x)\).
EX 4 Domain, codomain, range
\(f:\mathbb{R}\to\mathbb{R}\), \(f(x)=x^2\). Fill in domain, codomain, range.
Domain = \(\mathbb{R}\). Codomain = \(\mathbb{R}\).Range = \([0,\infty)\), because squares are never negative.
Natural domainIf no domain is stated, use all real \(x\) for which the formula is defined. Exclude zero denominators, negative inputs to even roots, and nonpositive inputs to logarithms.
EX 5 Four domains
Find domain and range: 1) \(\frac{1}{\sqrt{x}}\) 2) \(\sqrt[4]{x+1}\) 3) \(x^2+1\) 4) \(\frac{1}{\sqrt{x^2+3x+2}}\).
1) Need \(x\gt 0\). Domain \((0,\infty)\), range \((0,\infty)\).2) Need \(x+1\ge0\). Domain \([-1,\infty)\), range \([0,\infty)\).3) Domain \(\mathbb{R}\), range \([1,\infty)\).4) Need \((x+1)(x+2)\gt 0\). Domain \((-\infty,-2)\cup(-1,\infty)\), range \((0,\infty)\).

Combining Functions

Sums, products, quotients, and compositions of functions are defined only on inputs for which every expression involved is defined.

Composition rule\((g\circ f)(x)=g(f(x))\). The composition is valid on inputs \(x\in D_f\) such that \(f(x)\in D_g\); for all of \(D_f\), require \(R_f\subseteq D_g\).
EX 6 Which compositions exist?
\(f(x)=\frac{1}{x+1}\), \(g(x)=\sqrt{x}\), \(h(x)=x+1\). Test \(f\circ g\), \(g\circ f\), \(g\circ h\).
\(f\circ g\) exists: \(R_g=[0,\infty)\subseteq D_f=\mathbb{R}\setminus\{-1\}\). \((f\circ g)(x)=\frac{1}{\sqrt{x}+1}\).\(g\circ f\) does not exist: \(f\) can output negatives, but \(g\) needs input \(\ge0\).\(g\circ h\) does not exist on all \(\mathbb{R}\): \(h\) can output negatives.
TRY Compose
If \(f(x)=\sqrt{x-1}\) and \(g(x)=x^2+1\), find \(f\circ g\).
\(f(g(x))=\sqrt{(x^2+1)-1}=\sqrt{x^2}=\)\(|x|\).It exists for all real \(x\) because \(g(x)\ge1\).

Injective, Surjective, Inverse

A function is injective if distinct inputs give distinct outputs, surjective if its range equals its codomain, and bijective if both conditions hold. A bijection has an inverse function.

Injective
No two different inputs land on the same output. Test: \(f(x_1)=f(x_2)\) implies \(x_1=x_2\). Graph test: horizontal line hits at most once.
Surjective
Range = codomain. Every target value is hit by at least one input.
EX 7 Classify
Classify: 1) \(f:\mathbb{R}\to\mathbb{R},\ f(x)=x^3\) 2) \(g:\mathbb{R}\to[0,\infty),\ g(x)=x^2\) 3) \(h:[0,2\pi]\to[-1,1],\ h(x)=\sin x\) 4) \(k:[-\pi/2,\pi/2]\to[-1,1],\ k(x)=\sin x\).
1) Bijective: increasing and hits every real.2) Surjective, not injective: \(g(-2)=g(2)\).3) Surjective, not injective: \(\sin0=\sin\pi\).4) Bijective: sine maps distinct inputs to distinct outputs on this restricted interval and hits \([-1,1]\).
Inverse recipeWrite \(y=f(x)\), solve for \(x\), then swap letters. Domain and range swap.
EX 8 to 10 Inverses
Find inverses: 1) \(f(x)=2x+1\) 2) \(f(x)=\frac{x-1}{x+1}\) 3) \(f:[2,\infty)\to\mathbb{R},\ f(x)=x^2-4x+2\).
1) \(y=2x+1\Rightarrow x=\frac{y-1}{2}\), so \(f^{-1}(x)=\frac{x-1}{2}\).2) \(y(x+1)=x-1\Rightarrow x(y-1)=-(1+y)\), so \(f^{-1}(x)=\frac{1+x}{1-x}\), \(x\ne1\).3) \(f(x)=(x-2)^2-2\). Range \([-2,\infty)\). Since \(x\ge2\), take positive root: \(f^{-1}(x)=2+\sqrt{x+2}\).

Polynomials & Quadratics

A polynomial is a finite sum of terms \(a_kx^k\), where \(k\) is a nonnegative integer. A quadratic has degree \(2\); completing the square gives its vertex and range.

Polynomial\(P(x)=a_nx^n+\cdots+a_1x+a_0\), with \(n\) a nonnegative integer and \(a_n\ne0\). The degree is the highest nonzero power. A rational function is a quotient of polynomials where the denominator is not zero.
EX 14 Polynomial or not?
Classify: \(x^{33}-x^2+1\), \(x^2+\frac{1}{x^2}+1\), \(x^2+x+\sin x\).
Yes, degree 33.No, because \(\frac{1}{x^2}=x^{-2}\).No, because sin x is not a power of x.
EX 15 Complete the square
Write \(2x^2-x+3\) as \(k(x-a)^2+b\) and find the minimum.
\(2x^2-x+3=2(x^2-\frac12x)+3\).
Half of \(\frac12\) is \(\frac14\): \(2[(x-\frac14)^2-\frac{1}{16}]+3\).
\(2(x-\frac14)^2+\frac{23}{8}\).
Minimum is \(\frac{23}{8}\) at \(x=\frac14\).
DiscriminantFor \(ax^2+bx+c=0\), \(\Delta=b^2-4ac\). \(\Delta\gt 0\): two roots. \(\Delta=0\): repeated root. \(\Delta\lt 0\): no real roots.
EX 19 and 20 Remainder and factor theorem
1) Remainder when \(x^3+x^2+x+1\) is divided by \(x-1\). 2) Find \(k\) so \(x-k\) is a factor of \(3x^3-x^2-x-k\).

First two are rational. Third is not, because \(x^{3/2}\) is not a polynomial power.

Inequalities & Absolute Value

An inequality compares ordered quantities. Adding the same expression preserves the inequality, while multiplying or dividing by a negative number reverses the inequality sign. Absolute value gives distance on the real line.

Inequality ruleIf \(x\gt y\), then \(x+z\gt y+z\). If \(c\gt 0\), \(cx\gt cy\). If \(c\lt 0\), \(cx\lt cy\).
EX 21 True or false
\(1\le2\), \(1\le1\), \(-2\le-3\), \(0.999\ldots\lt 1\).
True, true, false, false.\(-2\) is greater than \(-3\). Also \(0.999\ldots=1\) exactly.
EX 22 Compound inequality
Solve \(x+2\lt 4x+3\lt 2x+20\).
Left: \(x+2\lt 4x+3\Rightarrow x\gt -\frac13\).Right: \(4x+3\lt 2x+20\Rightarrow x\lt \frac{17}{2}\).\(-\frac13\lt x\lt \frac{17}{2}\).
Absolute value\(|x-c|\) is the distance between \(x\) and \(c\). For \(a\gt0\), \(|x-c|\lt a\) means \(c-a\lt x\lt c+a\).
EX 28 Quick absolute value
Solve: \(|x|\lt 1\), \(|x|\le1\), \(|x|\gt 2\), \(|x+2|\lt 3\), \(|2x-5|\gt 4\).
\((-1,1)\), \([-1,1]\), \(x\lt -2\text{ or }x\gt 2\).\(|x+2|\lt 3\Rightarrow\)\(-5\lt x\lt 1\).\(|2x-5|\gt 4\Rightarrow2x-5\gt 4\) or \(2x-5\lt -4\Rightarrow\)\(x\gt \frac92\text{ or }x\lt \frac12\).
EX 29 and 30 Case work
1) \(|x+7|\lt 2x+10\). 2) \(|x-3|\ge |x+1|-1\).
1) \(-2x-10\lt x+7\lt 2x+10\) gives \(x\gt -\frac{17}{3}\) and \(x\gt -3\). So \(x\gt -3\).2) Equivalent to \(|x-3|-|x+1|\ge-1\). Split at \(-1\) and \(3\).\(x\lt -1\): value \(=4\), works. \(-1\le x\le3\): \(2-2x\ge-1\Rightarrow x\le\frac32\). \(x\gt 3\): value \(=-4\), fails.\(x\le\frac32\).

Exponents & Logarithms

Exponent laws define how powers with the same base combine. A logarithm is the inverse of exponentiation: \(\log_a b=x\) means \(a^x=b\).

Index laws\(a^na^m=a^{n+m}\), \((a^n)^m=a^{nm}\), \((ab)^n=a^nb^n\), \(a^{-n}=\frac{1}{a^n}\), \(a^{1/n}=\sqrt[n]{a}\).
EX 32 Solving powers
Find all real \(x\): \(x^2=4\), \(x^2=-1\), \(x^3=8\), \(x^3=-8\).
\(x=\pm2\); no real solution; \(x=2\); \(x=-2\).\(\sqrt4\) means the root \(2\), which is not negative, but \(x^2=4\) has two solutions.
EX 33 and 34 Same base thinking
1) If \(x\gt 0\) and \(x^2=8\), write \(x\) as \(2^k\). 2) Solve \(\frac{9^x}{3^{x+1}}=27\).
1) \(x=\sqrt8=(2^3)^{1/2}=\)\(2^{3/2}\).2) \(9^x=3^{2x}\) and \(27=3^3\). So \(3^{2x-(x+1)}=3^3\Rightarrow x-1=3\Rightarrow\)\(x=4\).
Log definition\(a^x=b\) iff \(x=\log_a b\). The base must satisfy \(a\gt0\) and \(a\ne1\), and the argument must satisfy \(b\gt0\).
EX 36 Read logs as questions
Compute \(\log_2 8\), \(\log_3 81\), \(\log_{1/2}2\).
\(2\) to what power is \(8\)? \(3\).\(3\) to what power is \(81\)? \(4\).\(\frac12\) to what power is \(2\)? \(-1\).
Log laws
  • \(\log_a(m^n)=n\log_a m\)
  • \(\log_a(mn)=\log_a m+\log_a n\)
  • \(\log_a\frac{m}{n}=\log_a m-\log_a n\)
  • \(\log_a b=\frac{\log_c b}{\log_c a}\)
EX 39 Change of base
Show \((\log_3 a)(\log_a9)=2\).
\(\log_a9=\frac{\log_3 9}{\log_3 a}\).\((\log_3a)\frac{\log_39}{\log_3a}=\log_39=\log_3(3^2)=\)\(2\).

Mathematical Induction

Mathematical induction proves statements indexed by integers. Prove a base case, then prove that \(P(k)\) implies \(P(k+1)\).

PMI recipe1) State \(P(n)\). 2) Prove base case. 3) Assume \(P(k)\). 4) Prove \(P(k+1)\). 5) Conclude for all \(n\) from the starting integer.
EX 50 Sum of integers
Prove \(\sum_{r=1}^{n}r=\frac{n(n+1)}{2}\) for all \(n\ge1\).
Base \(n=1\): LHS \(=1\), RHS \(=\frac{1(2)}{2}=1\).Assume \(\sum_{r=1}^{k}r=\frac{k(k+1)}{2}\).Then \(\sum_{r=1}^{k+1}r=\frac{k(k+1)}{2}+(k+1)=\frac{(k+1)(k+2)}{2}\).That is the formula with \(n=k+1\). Done.
EX 51 Sum of squares
Prove \(\sum_{r=1}^{n}r^2=\frac{n(n+1)(2n+1)}{6}\).
Base n=1: both sides =1.Assume true for \(k\). Add \((k+1)^2\):\(\frac{k(k+1)(2k+1)}{6}+(k+1)^2=\frac{(k+1)[k(2k+1)+6(k+1)]}{6}\).Inside \(=2k^2+7k+6=(k+2)(2k+3)\). So result = \(\frac{(k+1)(k+2)(2k+3)}{6}\), the formula for \(k+1\).
EX 52 and 53 Inequalities
1) Prove \(n!\gt 2^n\) for \(n\ge4\). 2) Bernoulli: for \(x\gt -1\) and \(x\ne0\), prove \((1+x)^n\gt 1+nx\) for \(n\ge2\).
1) Base \(n=4\): \(24\gt 16\). If \(k!\gt 2^k\) and \(k+1\gt 2\), then \((k+1)!=(k+1)k!\gt 2\cdot2^k=2^{k+1}\).2) Base \(n=2\): \((1+x)^2=1+2x+x^2\gt 1+2x\) because \(x^2\gt 0\).Assume \((1+x)^k\gt 1+kx\). Since \(1+x\gt 0\), multiply: \((1+x)^{k+1}\gt (1+kx)(1+x)=1+(k+1)x+kx^2\).If \(x\gt -1\) and \(x\ne0\), \(kx^2\gt 0\), so \((1+x)^{k+1}\gt 1+(k+1)x\).

\(f=(x+3)^2+1\), so range \([1,\infty)\).\(g=2(x+\frac32)^2+\frac92\), so \(g\) range \([\frac92,\infty)\).\(\frac1g\) is positive and at most \(\frac29\), approaching \(0\) but never \(0\). Range \((0,\frac29]\).
Q8 Absolute value
Solve \(|2x-5|=1\), \(|x+3|\ge7\), \(|x+1|\gt -1\).
\(2x-5=\pm1\Rightarrow\)\(x=2\text{ or }3\).Distance from \(-3\) at least \(7\): \(x\le-10\text{ or }x\ge4\).\(|x+1|\) is always \(\ge0\), and \(0\gt -1\), so all real \(x\).
Q9 Logs
Simplify \(\log_8 2\), \(\log_9 27\), \(4^{\log_2 3}\), \(\ln\sqrt{e}-\log_{100}10\).
\(\log_8 2=\frac{\log_2 2}{\log_2 8}=\)\(\frac13\).\(\log_9 27=\frac32\). \(4^{\log_2 3}=(2^2)^{\log_2 3}=\)\(9\).\(\ln\sqrt e=\frac12\) and \(\log_{100}10=\frac12\), so result \(0\).
Q10 Sigma notation
Write in sigma notation: odd numbers \(1\) to \(99\); \(\frac13+\frac19+\cdots+\frac{1}{243}\); \(\frac13+\frac18+\frac{1}{15}+\frac{1}{24}+\frac{1}{35}+\frac{1}{48}\).
\(\sum_{k=1}^{50}(2k-1)\).\(\sum_{k=1}^{5}\left(\frac13\right)^k\).Denominators \(3,8,15,24,35,48=k^2+2k=k(k+2)\), \(k=1,\ldots,6\). \(\sum_{k=1}^{6}\frac{1}{k(k+2)}\).
Q11 Trig equations
Solve \(\sin5\theta=\frac12\) and \(\sin4\theta=\frac{\sqrt3}{2}\) for \(-\pi\le\theta\le\pi\).
For \(\sin5\theta=\frac12\), let \(u=5\theta\), so \(-5\pi\le u\le5\pi\). \(u=\frac{\pi}{6}+2k\pi\) or \(u=\frac{5\pi}{6}+2k\pi\).\(\theta\in\{-\frac{23\pi}{30},-\frac{19\pi}{30},-\frac{11\pi}{30},-\frac{7\pi}{30},\frac{\pi}{30},\frac{\pi}{6},\frac{13\pi}{30},\frac{17\pi}{30},\frac{25\pi}{30},\frac{29\pi}{30}\}\).For \(\sin4\theta=\frac{\sqrt3}{2}\), \(u=4\theta\) in \([-4\pi,4\pi]\). \(u=\frac{\pi}{3}+2k\pi\) or \(u=\frac{2\pi}{3}+2k\pi\).\(\theta\in\{-\frac{11\pi}{12},-\frac{5\pi}{6},-\frac{5\pi}{12},-\frac{\pi}{3},\frac{\pi}{12},\frac{\pi}{6},\frac{7\pi}{12},\frac{2\pi}{3}\}\).
Q12 and Q13 Induction
Guess \(\sum_{k=m}^{\infty}r^k\) for \(|r|\lt 1\). Prove one: \(1+5+\cdots+(4n-3)=n(2n-1)\).
Geometric tail: \(\sum_{k=m}^{\infty}r^k=\frac{r^m}{1-r}\).Base \(n=1\): \(1=1(1)\).Assume sum to \(k\) is \(k(2k-1)\). Add next term \(4(k+1)-3=4k+1\).\(k(2k-1)+4k+1=2k^2+3k+1=(k+1)(2k+1)\), which is formula for \(k+1\).

Common mistakes
Domain first

Do not simplify away restrictions such as denominator \(=0\).

Inverse check

A function needs a one to one restriction before an inverse is a function.

Log input

\(\ln 0\) and \(\ln(\text{negative})\) are not real numbers.

Induction gap

The assumption is only \(P(k)\). You still have to prove \(P(k+1)\).

5 minute self check
1. What does \([a,b)\) mean?
All \(x\) with \(a\le x\lt b\). Include \(a\), exclude \(b\).
2. What is the domain of \(\sqrt{x-5}\)?
\(x\ge5\), so \([5,\infty)\).
3. When does \(\ln f(x)\) exist?
When \(f(x)\gt0\).
4. What condition makes \(\sum ar^k\) converge?
\(|r|\lt1\).
5. What are the two checks in induction?
Prove the base case, then prove \(P(k)\Rightarrow P(k+1)\).
